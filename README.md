## Descripción

Esta herramienta permite validar el usuario y la contraseña de un sitio de internet validando con el post de login. Está integramente escrito en Java.\n\r

Dependencias:\n\r
\tjavax.mail.jar
\tjsoup-1.8.3.jar
\tmysql-connector-java-5.0.8-bin.jar

## Base de Datos

La base de datos es MySQL y el script para la creación está en la carpeta "dbScript"

## Ejecución

La clase que contiene el "public static void main" se encuentra dentro de la carpeta "validator"
